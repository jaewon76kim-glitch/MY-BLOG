# Spec: 도플러 + 전파지연 위성 통과 시뮬레이터

## 1. 선택한 웹앱

**이름:** 위성 통과 도플러·RTT 시뮬레이터  
**한 줄 설명:** LEO/MEO/GEO 위성이 지평선에서 천정을 지나 반대쪽으로 통과하는 동안 도플러 천이(Hz)와 왕복 전파지연(RTT, ms)이 실시간으로 어떻게 변하는지 애니메이션으로 보여주는 인터랙티브 시뮬레이터

## 2. 핵심 기능 목록

1. **위성 궤도 애니메이션** — 캔버스 위에 위성이 지평선(仰角 0°)에서 최대 앙각(천정 통과 또는 최대 앙각 El_max)을 거쳐 반대편 지평선으로 이동하는 호(arc)를 SVG/Canvas로 그림
2. **실시간 도플러 천이 계산** — 현재 앙각(elevation)과 위성 속도(v)로부터 `f_d = (v/c) * cos(θ) * f_carrier` 를 매 프레임 계산하고 그래프에 실시간 플롯
3. **실시간 RTT 계산** — 현재 경사거리(slant range)를 `R = √(h² + R_E² - 2hR_E·sin(El)) - R_E (근사)` 공식으로 산출, `RTT = 2R/c` 변환 후 그래프에 표시
4. **파라미터 슬라이더** — 위성 궤도 고도(h: 400~35786km), 캐리어 주파수(f: 1~30GHz), 최대 앙각(El_max: 10~90°) 입력으로 계산 즉시 갱신
5. **일시정지·속도 배율 제어** — 시뮬레이션 재생/일시정지 버튼, 시간 배율(1×/5×/10×) 선택으로 통과 과정을 단계별 관찰 가능

## 3. 파일 구조

```
/apps/doppler-sim/
├── index.html          # 앱 진입점 (단일 페이지)
├── css/
│   └── style.css       # 앱 전용 스타일 (블로그 공통 CSS와 독립)
└── js/
    ├── sim.js          # 궤도·도플러·RTT 물리 계산 모듈
    ├── renderer.js     # Canvas 위성 애니메이션 렌더러
    └── chart.js        # Chart.js 래퍼 — 도플러·RTT 실시간 차트
```

## 4. 주요 UI 구성

### 입력 패널 (좌측 또는 상단)

| 컨트롤 | 범위 | 기본값 |
|---|---|---|
| 궤도 고도 h (km) 슬라이더 | 400 ~ 35786 | 550 (LEO) |
| 캐리어 주파수 f (GHz) 슬라이더 | 1 ~ 30 | 12 |
| 최대 앙각 El_max (°) 슬라이더 | 10 ~ 90 | 90 |
| 재생 / 일시정지 버튼 | — | 재생 |
| 속도 배율 선택 (×1 / ×5 / ×10) | — | ×5 |

### 출력 영역 (우측 또는 하단)

- **위성 통과 애니메이션 캔버스** — 지평선·지면·위성 아이콘·지상국 아이콘, 현재 앙각과 경사거리(km)를 오버레이로 표시
- **도플러 천이 실시간 그래프** — x축: 경과 시간(s), y축: f_d (kHz), 통과 중간(천정)에서 0 교차를 시각적으로 확인
- **RTT 실시간 그래프** — x축: 경과 시간(s), y축: RTT (ms), 최솟값(천정 통과 시)이 레이블로 표시
- **수치 요약 카드** — 현재 앙각, 경사거리, 도플러 천이, RTT를 숫자로 실시간 업데이트

## 5. 사용할 기술

- **순수 HTML5 / CSS3 / JavaScript (ES2020+)** — 빌드 도구 없음, `type="module"` 없음
- **Canvas API** — 위성 통과 애니메이션 렌더링 (`requestAnimationFrame`)
- **Chart.js 4.x (로컬 복사본)** — 도플러·RTT 실시간 라인 차트
  - 파일: `js/vendor/chart.umd.min.js` (CDN 금지, 로컬 배치)
- **수식은 JS로 직접 구현** — 외부 수식 렌더러 불필요 (KaTeX 등 배제)

## 6. 구현 범위 외 (이번 버전 제외)

- 3D 지구 구체 렌더링 (Three.js 등) — 2D 호(arc) 표현으로 대체
- 실제 TLE(Two-Line Elements) 파싱·실시간 위성 추적 — 단순 원형 궤도 근사 사용
- 다중 위성 동시 표시 (Starlink 군집 시뮬레이션)
- 대기 굴절·전리층 보정 계산
- 모바일 터치 드래그로 위성 위치 직접 조작
- 강우감쇠, 링크버짓 연동 계산
