## 2. 핵심 기능 목록

1. **토이 데이터셋 선택 + 실시간 결정경계 히트맵**: XOR, 원형(concentric circles), 나선형(2-spiral) 중 하나를 선택하면 2D 산점도가 캔버스에 그려지고, 학습이 진행되는 동안 배경에 현재 신경망의 분류 결정경계(확률 히트맵)가 실시간으로 갱신된다.
2. **네트워크 구조 설정**: 은닉층 뉴런 수(슬라이더, 예: 2~8개), 활성함수(시그모이드/tanh/ReLU) 선택. 값을 바꾸면 네트워크가 초기화되고 다이어그램(원+선)이 즉시 다시 그려진다.
3. **학습 컨트롤과 순전파/역전파 애니메이션**: 학습률 슬라이더, 학습모드(배치/확률적/미니배치) 선택, 재생/일시정지/한 스텝씩(step) 진행/리셋 버튼. 재생 중에는 네트워크 다이어그램 위에서 순전파(입력→출력, 파란색)와 역전파 오차신호(출력→입력, 빨간색, δ→γ→β 순서)가 애니메이션으로 흐르는 것을 보여준다.
4. **손실 곡선 & 지표 패널**: 에폭에 따른 손실(제곱오차/교차엔트로피) 변화를 Chart.js 라인 차트로 실시간 표시하고, 현재 에폭 수·정확도·기울기 소실 여부(각 층 평균 |δ| 값 비교)를 텍스트로 함께 표시한다.
5. **프리셋 시나리오 버튼**: "얕은 네트워크로 XOR 실패시키기", "은닉층 추가 후 XOR 성공", "시그모이드 vs ReLU로 기울기 소실 비교" 같은 원클릭 프리셋을 제공해, 사용자가 스스로 파라미터를 조합하지 않아도 핵심 교훈(선형분리 불가 → 은닉층으로 해결, 기울기 소실 문제)을 즉시 재현해볼 수 있게 한다.

## 3. 파일 구조

```
/apps/mlp-playground/
  index.html              # 레이아웃: 컨트롤 패널 + 데이터/결정경계 캔버스 + 네트워크 다이어그램 캔버스 + 손실 차트
  css/style.css           # 반응형 레이아웃(모바일 세로 스택), 기존 앱과 톤 통일
  js/network.js           # MLP 순전파/역전파 순수 JS 구현(행렬 연산, δ/γ/β 오차신호 계산, 활성함수 3종)
  js/datasets.js          # 토이 데이터셋 생성 함수(XOR, 원형, 나선형) + 정규화
  js/visualizer.js        # 캔버스 렌더링: 결정경계 히트맵, 데이터 산점도, 네트워크 다이어그램(노드/엣지), 순전파·역전파 신호 애니메이션
  js/charts.js            # Chart.js 손실 곡선 설정/갱신
  js/main.js              # 전체 상태 관리, 컨트롤 이벤트 바인딩, 학습 루프(requestAnimationFrame 기반), 프리셋 정의
  js/vendor/chart.umd.min.js   # 기존 apps/link-budget-calc(또는 doppler-sim)의 사본 재사용(Chart.js v4.4.4, 동일 파일)
```

## 4. 주요 UI 구성

- **좌측(또는 상단, 모바일에서는 스택) 컨트롤 패널**
  - 입력: 데이터셋 선택(라디오/드롭다운), 은닉층 뉴런 수(슬라이더), 활성함수(드롭다운), 학습률(슬라이더, 로그스케일), 학습모드(배치/SGD/미니배치), 재생/일시정지/스텝/리셋 버튼, 프리셋 버튼 3개.
- **중앙 캔버스 1 — 데이터 & 결정경계**
  - 출력: 2D 산점도(클래스별 색 구분) 위에 현재 네트워크의 분류 확률을 색상 그라디언트 히트맵으로 오버레이. 학습 진행에 따라 매 프레임(또는 매 N 에폭) 갱신.
- **중앙 캔버스 2 — 네트워크 다이어그램**
  - 출력: 입력층/은닉층/출력층 노드를 원으로, 가중치를 선(굵기·색으로 부호/크기 표현)으로 그림. 재생 중 순전파는 파란 입자가 좌→우로, 역전파 오차신호(δ, γ, β)는 빨간 입자가 우→좌로 흐르는 애니메이션.
- **하단 — 손실 곡선 그래프(Chart.js 라인차트)**
  - 출력: x축 에폭, y축 손실값. 정확도·현재 에폭·레이어별 평균 |오차신호| 텍스트 지표를 그래프 옆에 병기.

## 5. 사용할 기술

- 순수 HTML/CSS/JS (프레임워크 없음). MLP 순전파/역전파는 외부 라이브러리 없이 `js/network.js`에서 직접 구현(행렬은 일반 배열/이중배열로 충분한 규모).
- **Chart.js** (손실 곡선 전용): 기존 관례대로 CDN 대신 `js/vendor/chart.umd.min.js`에 로컬 복사. `apps/link-budget-calc/js/vendor/chart.umd.min.js` (또는 doppler-sim/constellation-sim, 세 곳 모두 동일한 Chart.js v4.4.4 UMD 빌드) 파일을 그대로 복사해 재사용 가능함을 확인했다. 신규 다운로드 불필요.
- 결정경계 히트맵과 네트워크 다이어그램, 순전파/역전파 애니메이션은 Chart.js가 아니라 `<canvas>` 2D API를 직접 사용(기존 doppler-sim의 `js/renderer.js` 패턴과 동일한 접근).
