## 2. 핵심 기능 목록

### 2.1 EIRP 계산 (송신측)
- 입력: 송신전력 $P_{t,\mathrm{dBW}}$, 송신 안테나 이득 $G_{T,\mathrm{dB}}$
- 수식(tex 905절): $\mathrm{EIRP}_{\mathrm{dBW}} = P_{t,\mathrm{dBW}} + G_{T,\mathrm{dB}}$

### 2.2 G/T 계산 (수신측)
- 입력: 수신 안테나 이득 $G_{R,\mathrm{dB}}$, 안테나 잡음온도 $T_{ant}$(K), 수신기 잡음지수 $F$(dB)
- 수식(tex 905절):
  - $T_e = T_0(F-1)$, $T_0=290\,$K
  - $T_{sys} = T_{ant} + T_e$
  - $(G/T)_{\mathrm{dB}} = G_{R,\mathrm{dB}} - 10\log_{10}T_{sys}$

### 2.3 자유공간 경로손실
- 입력: 슬랜트 거리 $d$(km, 궤도고도 h + 앙각 θ로부터 산출), 반송주파수 $f_c$(GHz)
- 수식(tex 148절, 231절에서 재인용): $L_{FS,\mathrm{dB}}(d) = 20\log_{10}d_{km} + 20\log_{10}f_{c,MHz} + 32.44$

### 2.4 ITU-R P.618 강우감쇠
- 입력: 앙각 θ, 강우고도 $h_R$(2~5km), 지상국 해발고도 $h_S$, 연간 0.01% 초과 강우강도 $R_{0.01}$(mm/h), 주파수 $f_c$(편파는 원형 편파로 고정, 단순화)
- 수식(tex 859절):
  - $L_S = (h_R-h_S)/\sin\theta$ (슬랜트 경로장)
  - $L_G = L_S\cos\theta$, $r = 1/(1+L_G/L_0)$ (경로 축소계수 — $L_0$는 $R_{0.01}$에 따른 ITU-R P.618 경험식으로 산출)
  - $L_E = L_S\cdot r$ (유효경로장)
  - $\gamma_R = k R_{0.01}^\alpha$ [dB/km] (ITU-R P.838 회귀계수 $k,\alpha$는 주파수 대역별 표로 참조 — tex는 함수형만 제시하므로 Build 단계에서 공식 ITU-R P.838 계수표 값을 그대로 가져다 쓰고 임의로 만들지 않는다)
  - $A_{0.01} = \gamma_R\cdot L_E$ [dB] → 강우마진(rain margin)으로 링크버짓에 반영

### 2.5 링크버짓 / Eₛ/N₀ 및 수신감도
- 입력: 심볼률 $R_s$, 요구 $(E_s/N_0)_{min}$(변조방식 프리셋 참고값), 급전선 손실 $L_{rf}$
- 수식(tex 231절 + 905절):
  - $\dfrac{E_s}{N_0}\Big|_{\mathrm{dB}} = \mathrm{EIRP}_{\mathrm{dBW}} + (G/T)_{\mathrm{dB}} - L_{p,\mathrm{dB}} - A_{rain} - (k_B)_{\mathrm{dB}} - R_{s,\mathrm{dB}} - L_{rf,\mathrm{dB}}$
    ($k_B=1.38\times10^{-23}$, $(k_B)_{\mathrm{dBm}}\approx-198.6\,$dBW/K/Hz)
  - 링크마진(Link Margin) $= E_s/N_0(\text{계산값}) - (E_s/N_0)_{min}$
  - 수신감도 $S_{rx,\mathrm{dBm}} = P_{t}+G_T+G_R-L_{p}$, 최대허용경로손실 $L_{p,max}=P_t+G_T+G_R-S_{rx}$ (보조 지표로 표시)

### 2.6 업링크 / 다운링크 전환
- 토글 버튼으로 방향 전환 (tex 905절 요약식): 다운링크는 $\mathrm{EIRP}_{sat}+(G/T)_{gw}$, 업링크는 $\mathrm{EIRP}_{gw}+(G/T)_{sat}$ 조합. 계산 로직은 동일하고, 방향에 따라 입력 슬라이더의 레이블("위성 EIRP"/"게이트웨이 EIRP" 등)과 기본값 프리셋만 바뀐다(위성은 저EIRP·저G/T, 게이트웨이는 고EIRP·고G/T인 비대칭성을 프리셋으로 반영).

### 2.7 아웃티지 확률
- 입력: 쉐도잉 표준편차 $\sigma_s$(dB, 지상 셀룰러 5~12dB vs 위성-지상 LMS 채널은 더 큰 값 가능하도록 슬라이더 범위 확장)
- 수식(tex 231절): $M_{shad}=$ 링크마진(dB), $O_N = Q(M_{shad}/\sigma_s)$ — $Q(\cdot)$는 표준정규 Q함수를 JS로 근사 구현(예: Abramowitz–Stegun 근사)

### 2.8 실시간 시각화
- 링크버짓 워터폴(막대) 그래프: EIRP → +G/T → −L_p → −A_rain → −k_B → −R_s → −L_rf 순서로 누적 막대를 그려 최종 Eₛ/N₀에 도달하는 과정을 시각화
- 게이지/바: Eₛ/N₀ 대비 요구 임계값과 링크마진을 시각적 막대(부족/충분 색상 구분)로 표시
- 강우감쇠 vs 강우강도 곡선: 현재 주파수·앙각 기준으로 $R_{0.01}$을 0~150mm/h까지 스윕한 $A_{0.01}(R)$ 그래프
- 아웃티지 확률 vs 마진 곡선: 현재 $\sigma_s$ 기준 $O_N$을 마진(dB) 함수로 그린 그래프, 현재 마진 위치를 점으로 표시
- 수치 요약 카드: EIRP, G/T, $L_{FS}$, $A_{rain}$, Eₛ/N₀, 링크마진, 아웃티지 확률을 실시간 숫자로 표시

### 출력 영역 (우측 또는 하단)

- **수치 요약 카드** — EIRP(dBW), G/T(dB/K), $L_{FS}$(dB), $A_{rain}$(dB), Eₛ/N₀(dB), 링크마진(dB, 양/음수 색상 구분), 아웃티지 확률(%)
- **링크버짓 워터폴 막대 그래프** — EIRP부터 시작해 각 손실/이득 항목을 순서대로 누적해 최종 Eₛ/N₀에 도달하는 과정을 막대로 시각화
- **마진 게이지 바** — 요구 임계값 대비 현재 Eₛ/N₀ 위치(충분/부족 색상)
- **강우감쇠 곡선** — x축: $R_{0.01}$(mm/h), y축: $A_{0.01}$(dB), 현재 설정값을 점으로 표시
- **아웃티지 확률 곡선** — x축: 마진(dB), y축: $O_N$(로그 스케일 또는 %), 현재 마진 위치를 점으로 표시

## 5. 사용할 기술

- **순수 HTML5 / CSS3 / JavaScript (ES2020+)** — 빌드 도구 없음, `type="module"` 없음, doppler-sim과 동일한 관례
- **Chart.js 4.x (로컬 복사본)** — `js/vendor/chart.umd.min.js`, `apps/doppler-sim`에서 그대로 복사해 재사용(CDN 금지)
- **수식은 JS로 직접 구현** — `linkbudget.js`, `rain.js`에 tex 원문의 수식을 그대로 옮김. ITU-R P.838(k, α)과 ITU-R P.618($L_0$ 경험식) 수치표/계수는 공식 ITU-R 권고안 값을 그대로 사용하고 임의로 만들지 않는다(Build 단계에서 출처를 코드 주석에 명시)
- **Q함수(정규분포 우측꼬리)** — 외부 통계 라이브러리 없이 Abramowitz–Stegun 근사식으로 JS 자체 구현
- 외부 수식 렌더러(KaTeX 등) 불필요 — 계산 결과는 숫자/그래프로만 표시, 수식 표기는 UI 텍스트 설명으로 대체

## 6. 구현 범위 외 (이번 버전 제외)

- 위성 궤도 애니메이션(2D/3D) — `apps/doppler-sim`과 중복이므로 제외, 앙각/고도는 슬라이더 입력값으로만 취급
- 실시간 TLE 기반 위성 추적, 실제 위성 스펙 DB 연동
- ITU-R 강우 지역 지도(위경도 기반 $R_{0.01}$ 자동조회) — 수동 슬라이더/프리셋으로 대체
- 다중 반송파 간섭(C/I), 인접채널 간섭, 상호변조(IMD) 계산
- 적응부호변조(ACM) 시뮬레이션, MODCOD 자동 선택 로직
- 안테나 방사패턴(사이드로브 포함) 3D 시각화 — 이득은 단일 dBi 숫자로만 취급
- 편파(원형/선형) 별 ITU-R P.838 계수 차이의 세밀한 반영 — 원형 편파 기준 단순화
- 그림자 라이시안(shadowed-Rician), Loo 모델 등 소규모 페이딩 통계 기반 정밀 아웃티지 — 이번 버전은 로그노멀 쉐도잉 Q함수 근사만 사용
