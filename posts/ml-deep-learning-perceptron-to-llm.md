[지난 편](post.html?slug=ml-classical-algorithms-bayes-to-svm)에서 베이즈 정리부터 SVM까지 고전 머신러닝을 훑었다. 이번 편은 그 뒤를 잇는 신경망·CNN·RNN·트랜스포머·LLM이다. 흥미롭게도 이 절반은 앞의 절반과 끊어져 있지 않다 — 다층 퍼셉트론이 극복한 문제(XOR)는 SVM이 이미 커널트릭으로 풀었던 문제였고, ResNet의 잔차연결·트랜스포머의 셀프어텐션·RLHF의 보상모델은 각각 앞 편에서 다룬 개념(기울기 소실, 내적 유사도, 로지스틱회귀)의 재활용이다.

## 1. 퍼셉트론과 역전파 — 신경망의 탄생

**문제**: 뉴런 하나를 흉내낸 퍼셉트론(선형 분류기)은 XOR처럼 선형분리가 불가능한 데이터를 다룰 수 없다(1969년 민스키·페퍼트가 증명, 1차 AI 겨울의 원인). 앞 편의 커널트릭과 같은 발상 — 비선형 변환으로 선형분리가 쉬운 공간으로 옮기기 — 을 층을 쌓아 신경망 스스로 학습하게 하면 이 한계를 넘을 수 있다.

**키워드**: 활성함수, 다층 퍼셉트론(MLP), 오차역전파법(backpropagation), 연쇄법칙, 기울기 소실/폭발, ReLU

**핵심 수식**: 퍼셉트론은 `y = f(w·x)`. 은닉층을 거치는 MLP는
```
y = f(W⁽²⁾ f(W⁽¹⁾x))
```
가중치를 학습하려면 손실함수 `J = ½‖ŷ − y‖²`를 각 층의 가중치로 미분해야 하는데, 연쇄법칙을 출력층에서부터 거꾸로 적용하면 놀랍도록 재귀적인 패턴이 나온다 — 오차 신호 `δ`가 출력층에서 입력층 방향으로 가중치를 타고 전파된다:
```
∂J/∂w⁽³⁾ = δ·u      (δ = (ŷ−y)·f'(·))
∂J/∂w⁽²⁾ = γ·z      (γ = f'(·) · Σ δ·w⁽³⁾)
```
층이 깊어질수록 `f'(·)`(시그모이드는 최대 0.25)가 계속 곱해져 입력에 가까운 층의 기울기가 지수적으로 사라진다 — **기울기 소실**. 미분값이 항상 1인 `ReLU(x)=max(0,x)`가 이 문제를 크게 완화하면서 딥러닝이 실제로 학습 가능해졌다.

## 2. CNN — 이미지의 지역 구조를 활용하기

**문제**: 완전연결층을 이미지에 그대로 쓰면 파라미터가 폭증한다(200×200 이미지에 은닉뉴런 100개만 둬도 400만 파라미터). 이미지는 가까운 픽셀끼리만 강하게 상관되어 있다는 구조를 완전연결은 전혀 활용하지 못한다.

**키워드**: 부분연결, 가중치 공유, 합성곱(convolution), 특징맵, 패딩/스트라이드/풀링, LeNet-5, AlexNet(ReLU+드롭아웃), VGG(작은 커널 여러 겹), 1×1 합성곱, ResNet(잔차연결)

**핵심 수식**: 같은 커널(필터) `W`를 이미지 전체에 슬라이딩하며 적용해 파라미터 수를 커널 크기로 고정한다. 출력 특징맵 크기는
```
n' = ⌊(n + 2p − h)/s⌋ + 1     (n:입력, h:커널, s:스트라이드, p:패딩)
```
층을 아주 깊게 쌓으면 CNN도 기울기 소실이 재발한다. **ResNet**은 몇 개 층을 건너뛰어 입력을 출력에 더하는 잔차연결로 해결한다:
```
x₄ = F(x₁,x₂,x₃) + x₁
∂x₄/∂x₁ = (곱셈 경로, 소실 가능) + 1     (덧셈 지름길이 최소 1을 보장)
```

## 3. RNN과 LSTM — 순서 있는 데이터 다루기

**문제**: MLP·CNN은 입력 크기가 고정돼 있고 순서를 자연스럽게 다루지 못한다("개가 사람을 물었다" ≠ "사람이 개를 물었다"). CNN의 가중치 공유를 공간이 아니라 **시간**축에 적용하면 임의 길이의 시퀀스를 처리할 수 있다.

**키워드**: 순환 신경망(RNN), 은닉상태, 시간을 통한 역전파(BPTT), 장기 의존성, LSTM(셀 상태, 게이트), GRU, 단어 임베딩, 언어모델링

**핵심 수식**: 같은 가중치를 모든 시점에 반복 적용한다.
```
h_t = f(W_hh·h_(t-1) + W_xh·x_t + b_h)
```
시퀀스가 길어지면 `∂h_T/∂h_1`이 `T-1`개 행렬의 곱이 되어 기울기가 소실되거나 폭발한다. **LSTM**은 ResNet과 같은 원리(곱셈 대신 덧셈)를 시간축에 적용한다 — 셀 상태를 원소별 덧셈으로 갱신해 기억을 보존한다:
```
c_t = f_t ⊙ c_(t-1) + i_t ⊙ c̃_t     (f_t: 잊기 게이트, i_t: 입력 게이트)
```

## 4. 어텐션과 트랜스포머 — 병목을 없애고 병렬화하기

**문제**: 인코더-디코더(Seq2Seq) 구조는 문장 전체 정보를 고정 크기 벡터 하나에 압축해야 하는 **병목**이 있다. 게다가 RNN은 `h_t`가 `h_(t-1)`을 먼저 필요로 해 병렬 계산이 안 된다.

**키워드**: 어텐션, 맥락벡터, 셀프 어텐션, Query/Key/Value, 스케일링, 멀티헤드 어텐션, 위치 인코딩, 잔차연결+층정규화, BERT(인코더-온리) vs GPT(디코더-온리, 인과적 마스킹)

**핵심 수식**: 각 위치가 다른 모든 위치와 얼마나 관련 있는지를 내적(SVM 커널트릭과 같은 발상)으로 재고, 소프트맥스로 정규화해 value를 가중합한다.
```
Attention(Q,K,V) = softmax(QKᵀ/√d_k) · V
```
`√d_k`로 나누는 건 차원이 커질수록 내적의 분산이 커져 소프트맥스가 극단화되는 걸 막기 위해서다. 이 계산은 모든 위치에 대해 동시에 할 수 있어 RNN의 순차적 제약과 장기 의존성 문제를 한 번에 해결한다. GPT는 미래 위치를 보지 못하게 마스킹한 디코더만 써서 "다음 토큰 예측"을 병렬로 학습한다.

## 5. LLM 시대 — 스케일업, 정렬, 효율화

**문제**: 트랜스포머를 어떻게 키우고, 무엇으로 학습시키고, 사람이 원하는 방향으로 다듬을 것인가.

**키워드**: 사전학습-파인튜닝, 스케일링 법칙(Kaplan/Chinchilla), 인-컨텍스트 러닝, 창발적 능력, RLHF(SFT→보상모델→PPO), DPO, LoRA, MoE, 테스트 시점 계산, RAG

**핵심 수식**: 손실 `L`이 모델 크기 `N`, 데이터량 `D`에 대해 거듭제곱 법칙을 따른다(스케일링 법칙) — 작은 모델 몇 개로 큰 모델의 성능을 예측할 수 있다.
```
L(N) ≈ (N_c/N)^α_N,   L(D) ≈ (D_c/D)^α_D
```
사람의 선호(비교 판단)로 보상모델을 학습하는 단계는 앞 편의 로지스틱회귀와 형태가 완전히 같다:
```
P(y_w ≻ y_l | x) = σ(r_φ(x,y_w) − r_φ(x,y_l))
```
모델 전체를 파인튜닝하기엔 너무 크므로, **LoRA**는 가중치 변화량만 저랭크 행렬 곱으로 근사한다(파라미터 수 `d×k` → `r(d+k)`, `r≪d,k`):
```
W = W₀ + BA,   B∈ℝ^(d×r), A∈ℝ^(r×k)
```
**MoE**는 토큰마다 일부 전문가만 활성화해(`TopK` 라우팅) 총 파라미터는 키우면서 실제 연산량은 낮게 유지한다.

## 마치며

퍼셉트론의 XOR 한계를 다층으로 극복한 발상이 커널트릭의 재탕이었듯, ResNet의 잔차연결과 LSTM의 게이트는 같은 원리(곱셈 경로의 기울기 소실을 덧셈 경로로 우회)의 두 가지 구현이었다. 셀프 어텐션의 유사도 계산은 SVM의 커널트릭과 같은 내적이었고, RLHF의 보상모델은 로지스틱회귀 그 자체였다. 결국 1부에서 6부까지 다진 도구(확률, 최적화, 경사하강법, 내적 기반 유사도)가 형태만 바꿔가며 CNN·RNN·트랜스포머·LLM 전체를 관통한다는 게, 이 교재를 정리하며 가장 인상 깊었던 부분이다.
