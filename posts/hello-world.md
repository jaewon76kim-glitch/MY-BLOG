## 블로그를 시작하며

이 블로그는 **어떤 프레임워크도 사용하지 않고** 순수 HTML, CSS, JavaScript만으로 만들어졌습니다.

빌드 도구도 없고, npm도 없습니다. 파일을 열거나 간단한 정적 서버 하나면 바로 동작합니다.

## 왜 프레임워크 없이?

개발 환경을 간단하게 유지하고 싶었습니다. 블로그는 결국 글을 읽는 공간이고, 그 목적에 집중하면 복잡한 도구가 필요하지 않습니다.

> 단순함은 궁극의 정교함이다. — 레오나르도 다빈치

## 주요 기능

- 마크다운 파일로 글 작성
- 다크 모드 지원 (시스템 설정 + 수동 토글)
- 모바일 반응형 레이아웃
- 태그 필터 및 실시간 검색
- 포스트 내 목차(TOC) 자동 생성
- 읽기 시간 표시
- 코드 신택스 하이라이팅
