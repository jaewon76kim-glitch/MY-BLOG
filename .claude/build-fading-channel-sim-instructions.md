## 물리/수학 (theory.js, fading.js)

### 정의

- 순시 SNR `γ`, 평균 SNR `γ̄`(슬라이더로 조절, dB → 선형 변환해 사용).
- K-factor(라이시안): `K = A²/(2σ²)` (LOS 전력 대 산란전력 비). dB 슬라이더 → 선형 변환.
- m 파라미터(나카가미): `m ≥ 0.5`.
- K↔m 근사 변환: `m ≈ (K+1)² / (2K+1)` (표시용 참고값으로만 사용, 서로 역변환도 필요하면 이 식을 m에 대해 K로 근사적으로 풀거나 이분탐색으로 구한다).

### 이론 PDF (정규화 SNR γ/γ̄ 기준으로 그린다)

```
레일리:   f(γ) = (1/γ̄) exp(-γ/γ̄)                                    // 지수분포
라이시안: f(γ) = (K+1)/γ̄ * exp(-K) * exp(-(K+1)γ/γ̄) * I0(2*sqrt(K(K+1)γ/γ̄))
나카가미: f(γ) = (1/Γ(m)) * (m/γ̄)^m * γ^(m-1) * exp(-mγ/γ̄)
```

K=0 → 라이시안 식이 레일리와 정확히 일치해야 한다(I0(0)=1). m=1 → 나카가미가 레일리(지수분포)와 정확히 일치해야 한다.

### 이론 아웃티지 확률 `O(γ_th) = Pr[γ < γ_th]`

```
레일리:   O = 1 - exp(-γ_th/γ̄)
나카가미: O = P(m, m*γ_th/γ̄)     // 정규화 하위 불완전감마함수
라이시안: O = 1 - Q1(sqrt(2K), sqrt(2(K+1)*γ_th/γ̄))   // Marcum Q-함수(1차)
```

### 직접 구현해야 하는 특수함수 (외부 라이브러리 없이 순수 JS)

- **Bessel I0(x)**: 급수 `I0(x) = Σ_{k=0}^∞ (x/2)^(2k) / (k!)²`. x가 커지면(대략 x>20) 점근식
  `I0(x) ≈ exp(x)/sqrt(2πx)`을 병행해 오버플로를 피한다(라이시안 PDF에서 exp(-K)*I0(...)를 함께 계산할 때
  지수 폭주를 막기 위해, 가능하면 `exp(x - K)` 형태로 지수부를 미리 합쳐서 계산할 것 — K와 x가 모두 커도
  안정적으로 나오는지 확인).
- **Γ(m) (감마함수)**: Lanczos 근사. 정수/반정수 m은 `Γ(n)=(n-1)!`, `Γ(n+0.5)` 재귀식으로 정확히 계산 가능하면
  그 경로를 우선 사용해도 됨.
- **정규화 하위 불완전감마함수 P(m, x)**: 급수전개(x가 작을 때) + 연분수 또는 상보식(x가 클 때)의 표준
  근사(Numerical Recipes류 `gammainc` 구현 패턴 참고). 0~1 범위를 벗어나지 않는지 항상 검증.
- **Marcum-Q 함수 Q1(a,b)**: 급수전개 `Q1(a,b) = exp(-(a²+b²)/2) * Σ_{k=0}^∞ (a/b)^k I_k(ab)` (I_k는 k차
  Bessel, k=0,1만 있으면 재귀식 `I_{k+1}(x) = I_{k-1}(x) - (2k/x)I_k(x)`로 확장 가능) 또는 수치적분
  `Q1(a,b) = ∫_b^∞ x exp(-(x²+a²)/2) I0(ax) dx`을 사다리꼴/심슨 공식으로 직접 적분해도 된다(구현 단순성
  우선, 정확도는 소수점 2~3자리면 충분).

### 채널 생성 (fading.js) — 시간축 포락선

Clarke/Jakes sum-of-sinusoids 모델로 두 개의 독립 저역통과 가우시안 프로세스를 만든다:

```
N = 16~32 (오실레이터 개수)
X(t) = sqrt(2/N) * Σ_{n=1}^{N} cos(2π f_D cos(α_n) t + φ_n)
Y(t) = sqrt(2/N) * Σ_{n=1}^{N} sin(2π f_D cos(α_n) t + φ_n)   // X와 다른 각도 세트 α'_n 사용해 독립성 확보
α_n = (2πn - π + θ) / (4N),  θ는 랜덤 오프셋
φ_n = Box-Muller 또는 균일분포 [0,2π) 랜덤 위상 (시뮬레이션 시작 시 1회 생성 후 고정)
```

라이시안(LOS 있음)이면 `X(t) += sqrt(2K/(K+1))`, 스케일은 전체 평균전력이 목표 `γ̄`가 되도록 정규화한다.
레일리는 K=0(LOS 성분 없음)에 해당. 포락선 `R(t) = sqrt(X(t)² + Y(t)²)`, 순시 정규화 SNR `γ(t)/γ̄ = R(t)²`
(평균이 1이 되도록 정규화했을 때).

나카가미-m은 물리적으로 Clarke 모델의 자연스러운 결과가 아니므로, 감마분포 기반 직접 샘플링으로 만든다:
정규화 SNR 샘플을 `γ/γ̄ ~ Gamma(shape=m, scale=1/m)`에서 뽑고(감마 샘플링은 Marsaglia-Tsang 알고리즘 등
표준 방법 사용), 시간축 매끄러움은 저역통과 필터(예: 지수이동평균, 시상수는 `f_D`에 반비례)를 걸어
흉내낸다(완전히 물리적으로 정확한 시간상관은 아니어도 됨 — PDF와 아웃티지 정합성이 더 중요).

## 완료 조건 (수치 검증)

- K=0일 때 라이시안 이론 PDF가 레일리 이론 PDF와 곡선이 겹쳐야 한다(수식상 당연하지만 코드로도 그렇게 나오는지 확인).
- m=1일 때 나카가미 이론 PDF가 레일리(지수분포) PDF와 일치해야 한다.
- 충분히 누적된 몬테카를로 히스토그램의 평균이 설정한 `γ̄`(정규화 기준 1.0)에 가까워야 한다(±10% 이내).
- γ_th = γ̄(0dB)일 때 레일리 이론 아웃티지 ≈ `1 - e^{-1} ≈ 0.632`가 나오는지 확인.
- 브라우저에서 index.html을 열었을 때 바로 동작(정적 서버로 확인).
- 모바일 화면(375px)에서 레이아웃이 깨지지 않음.
