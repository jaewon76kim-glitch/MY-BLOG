## 상수 (semiconductor-constants.js에 정의)

```js
var E_G = 1.12;              // 실리콘 띠간격, eV
var N_C_300 = 2.8e19;        // 전도띠 유효상태밀도 @300K, cm^-3
var N_V_300 = 1.04e19;       // 가전자띠 유효상태밀도 @300K, cm^-3
var EPS_SI = 1.036e-12;      // 실리콘 유전율 ε = ε_r*ε0, F/cm (ε_r=11.7)
var Q = 1.602e-19;           // 전자전하, C
var K_B_EV = 8.617e-5;       // 볼츠만 상수, eV/K
var MU_N = 1350;             // 전자 이동도 @300K, cm^2/(V·s)
var MU_P = 480;              // 정공 이동도 @300K, cm^2/(V·s)
var TAU_N = 1e-6;            // 전자 소수캐리어 수명, s
var TAU_P = 1e-6;            // 정공 소수캐리어 수명, s
```

`N_c(T)`, `N_v(T)`는 온도 의존성 `N_c(T) = N_C_300 * (T/300)^1.5` (N_v도 동일 형태)로 스케일링한다
(원문의 `N_c ∝ T^{3/2}` 공식을 그대로 사용). `D_n = (k_BT/e)*MU_N`, `D_p = (k_BT/e)*MU_P`
(아인슈타인 관계식, 원문 그대로). `L_n = sqrt(D_n*TAU_N)`, `L_p = sqrt(D_p*TAU_P)`.

## 수학 (semiconductor-math.js에 순수 함수로 구현, 다른 모듈에서 재사용)

### 1) n_i(T), V_bi (섹션 1: 컨트롤 패널)

```
n_i(T) = sqrt(N_c(T) * N_v(T)) * exp(-E_G / (2 * k_B_eV * T))
V_bi(N_A, N_D, T) = (k_B_eV * T) * ln(N_A * N_D / n_i(T)^2)
```

- 검산: T=300K에서 `n_i(300) ≈ 6.6×10⁹ ~ 1×10¹⁰ cm⁻³` 범위(원문의 "n_i≈10^10" 진술과 자릿수가
  일치해야 함 — 정확히 이 상수들로 계산하면 약 6.6×10⁹이 나오는데, 이는 원문이 말하는 "~10^10 수준"에
  부합하는 정상 결과다. 계산값이 이 범위를 크게 벗어나면(예: 10^5 이나 10^15) 공식 구현에 버그가 있는
  것이므로 재확인할 것).
- 검산: N_A=1e17, N_D=1e15 cm⁻³, T=300K일 때 `V_bi`가 원문이 말하는 "전형적인 실리콘 다이오드
  0.6~0.7V" 범위에 들어오는지 확인. 도핑을 올리면(N_A, N_D 둘 다 1e18 이상) V_bi가 커지고, 낮추면
  (1e14 근처) V_bi가 작아지는 단조 증가 경향을 확인.

### 2) 공핍층 (섹션 3)

```
W(V_bi, V, N_A, N_D) = sqrt( (2*EPS_SI*(V_bi - V)/Q) * (1/N_A + 1/N_D) )
x_p = W * N_D / (N_A + N_D)     // N_A*x_p = N_D*x_n, x_p+x_n=W 로부터 유도
x_n = W * N_A / (N_A + N_D)
E_max = Q * N_A * x_p / EPS_SI  // = Q*N_D*x_n/EPS_SI 와 같아야 함(두 표현이 일치하는지 코드에서 확인)
```

- 전기장 프로파일: `E(x) = -Q*N_A/EPS_SI*(x+x_p)` (`-x_p<x<0`), `E(x) = -Q*N_D/EPS_SI*(x_n-x)`
  (`0<x<x_n`), 그 외 구간은 0. `x=0`에서 두 식이 연속(같은 값)인지 확인.
- 전위 프로파일 φ(x)는 E(x)를 적분(부호 주의: `E=-dφ/dx`)해 두 포물선을 이어붙인 곡선으로 구현하고,
  φ(x_n)-φ(-x_p) = V_bi-V가 되는지 검산.
- 검산: `N_A*x_p ≈ N_D*x_n` (전기적 중성 조건)이 항상 성립하는가.
- 검산: 비대칭 도핑(예: N_A=1e18, N_D=1e15)일 때 `x_n ≫ x_p`(저농도 쪽으로 공핍층이 훨씬 넓게
  형성됨)가 되는지 확인. 대칭 도핑(N_A=N_D)일 때는 `x_p=x_n=W/2`가 되는지 확인.
- 검산: 도핑을 높일수록(N_A, N_D 둘 다 증가) W가 감소하는 단조 감소 경향을 확인(원문 830행 명시적
  진술).

### 3) 쇼클리 다이오드 방정식 (섹션 4)

```
p_n0(N_D, T) = n_i(T)^2 / N_D
n_p0(N_A, T) = n_i(T)^2 / N_A
I0(N_A, N_D, T, A) = Q * A * ( D_p(T)*p_n0/L_p(T) + D_n(T)*n_p0/L_n(T) )
I(V) = I0 * (exp(Q*V/(k_B_eV*Q*T)) - 1)   // 주의: 지수 e^{eV/k_BT}에서 k_BT는 eV 단위이므로
                                            // 실제 구현은 exp(V / (K_B_EV*T)) - 1 (V는 볼트, K_B_EV*T는 eV=V 단위이므로 그대로 나눔)
```

**단위 주의**: `K_B_EV`는 eV/K 단위이므로 `K_B_EV*T`는 eV 값이며, 전압 V(볼트)와 같은 수치 스케일로
바로 나눌 수 있다(1eV의 에너지를 1V로 이동한 전자가 얻으므로 `eV/k_BT`의 `e`가 분자·분모에서 상쇄되어
`V/(K_B_EV*T)`만 남는다). 상온(T=300K)에서 `K_B_EV*T ≈ 0.02585V`(≈26mV, 원문이 명시한 값)가 되는지
직접 확인할 것 — 이 26mV 근사가 틀리면 지수함수 전체가 틀어진다.

- 검산: T=300K에서 `K_B_EV*300 ≈ 0.02585` (원문의 "≈26mV, 상온" 진술과 정확히 일치해야 함).
- 검산: V≪0(역방향)일 때 `I → -I0`로 포화되는지 확인.
- 검산: V가 `K_B_EV*T`(약 26mV)보다 훨씬 크면(예: V=0.5) I가 지수적으로 폭증(I0 대비 수십~수백
  자릿수 차이)하는지 확인.
- 검산: 도핑을 낮출수록(N_A, N_D 감소) `p_n0`, `n_p0`가 커져 `I0`가 커지고, 따라서 같은 V에서 순방향
  전류가 커지는 경향을 확인.

## UI 상세 (spec 4절 요약, 세부 구현은 자유)

- **상단 컨트롤 패널(고정/sticky)**: `N_A`, `N_D`(로그 슬라이더, 대략 10^14~10^19 cm⁻³, 값을
  "3.2×10¹⁷ cm⁻³" 형태로 표시), `T`(약 250K~450K), `V`(대략 -1V~+0.7V), 단면적 `A`(선택 조절, 기본값
  예: 1×10⁻⁴ cm²). 계산된 `V_bi`, `n_i(T)`를 항상 상단에 표시.
- 섹션 1(밴드 다이어그램): 캔버스, 접합 전/후 토글, `E_c`/`E_v`/`E_i`/`E_F` 색 구분, 밴드 휨
  `e(V_bi-V)` 화살표+숫자.
- 섹션 2(공핍층 프로파일): ρ(x)/E(x)/φ(x) 3단 그래프(세로 배치), `W`/`x_p`/`x_n`/`E_max` 숫자 병기.
- 섹션 3(I-V 곡선): Chart.js 선 그래프, 현재 V 동작점 마커, Y축 선형/로그 토글, `I0` 숫자 표시.
- 섹션 4(도핑 프리셋): "대칭 접합"(N_A=N_D=1e17), "비대칭 p+n 접합"(N_A=1e18, N_D=1e15), "저농도
  접합"(N_A=N_D=1e15) 등 3개 이상. 클릭 시 컨트롤 패널 슬라이더 값이 바뀌고 섹션 1~3이 함께 갱신되며,
  프리셋별 `V_bi`/`W`/`I0` 요약 표를 병기.
- 상단 앵커 네비게이션(모바일: 드롭다운/가로 탭) — 기존 앱과 동일 톤.
- 모바일 대응: 컨트롤 패널은 접이식(collapsible), 3단 그래프는 좁은 화면에서도 세로 스택 유지.

## 완료 조건 (수치 검증, 직접 확인)

- `K_B_EV*300 ≈ 0.02585`(≈26mV)로 정확히 계산되는가.
- `n_i(300)`이 6×10⁹~1×10¹⁰ cm⁻³ 범위로 나오는가(원문 "~10^10" 진술과 자릿수 일치).
- N_A=1e17, N_D=1e15, T=300K에서 `V_bi`가 0.6~0.7V 범위로 나오는가.
- 도핑을 높일수록 `V_bi` 증가, `W` 감소가 실제로 관찰되는가.
- 비대칭 도핑(N_A≫N_D)일 때 `x_n ≫ x_p`, 대칭 도핑일 때 `x_p=x_n=W/2`가 정확히 나오는가.
- `E(x)`가 `x=0`에서 두 구간의 식이 연속(같은 값)인가.
- V≪0일 때 `I→-I0`, V≫26mV일 때 `I`가 지수적으로 폭증하는가.
- 브라우저에서 index.html을 열었을 때 바로 동작.
- 모바일 화면(375px)에서 레이아웃이 깨지지 않음.
